Scratch Project, August 2019-September 2019, Computer Science Principles

The objective of this project was to create an interactive story using Scratch. Our story was based on Spider-Man, where he worked to defeat villains and come back home. In this project I did the majority of coding, and about a quarter of the documentation. Features of this project include: choices that create different storylines, sound effects,unique graphics and animations, and humorous endings. By the time it was due, our project didn't work as well as we had hoped. This was due to small bugs which would leave some sprites on the screen longer than needed, and some choices not making sense. However, if we had more time and worked more efficiently, this story would've developed into something very interesting and creative. Since this was my first ever computer science related project, I had a lot to learn. I learned how to use Scratch, and how to create variables and connect blocks. Using different costumes versus sprites was difficult for me, but this project was a great learning experience for that. It was also fun to get creative with the storyline and sprites. Instead of using the Scratch sprites to limit our options, we used PNG files from Google so we had more freedom. Making buttons was also a challenge because each button had to give a signal and it was difficult to keep track of each signal and where it went. My team also had some different views on where the storyline should go, so a lot of compromise was made. You can find our full project documentation here.

SCRUM Hangman Project, February 2020, Computer Science Principles

The objective of this project was to learn how to use the SCRUM method to create a hangman game using Python programming. This project was really fun because my team worked together really well. We divided up our work efficiently. While some programmed, others worked on SCRUM documentation. We also did some research on how to make the project look good, like using different colored text. It was confusing at first, but through trial and error we figured it out. I learned a lot about functions and loops. This project was also really helpful for learning how to split up work evenly and for learning how to research. We didn't really face any challenges, the entire project went smoothly.

Python Escape Room Project, January 2020, Computer Science Principles

The goal of this project was to code a game using Python. We had to make different choices and storylines using the theme of an escape room. Our creative process was to make something relatable to the players, like school. We then tried to come up with funny scenarios and interesting storylines that we could build on. It took a lot of planning out, and we created a flow chart to help us. In this project, my main role was documentation and coming up with the storyline. Some skills I learned were how to code using python and how ot connect functions. It was a function based project, so it was really good practice. One of the challenges we overcame was the issue of organization. There were a LOT of functions and we had to figure out how to keep track of all of them.
